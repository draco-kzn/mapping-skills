# 阶段03：Startup深度拆解

## 目标

逐家公司拆解，找出最有价值的创业公司，按统一格式横向对比。

## 步骤

### 3.1 确定对标分类

把Startup按以下维度分类：
- **产品类型**：ROV / AUV / 仿生平台 / 具身智能 / 特种机器人
- **技术路线**：液压 / 电动 / 仿生 / 具身AI
- **商业化阶段**：已有收入 / 意向订单 / 样机 / 科研

### 3.2 12列信息指标

每个Startup都要填以下12列信息：
1. **公司名称**（中英文）
2. **细分赛道**（一句话定位）
3. **成立时间/总部**
4. **创始人/CEO**（背景/年龄/履历亮点）
5. **技术路线**（核心技术/路线选择）
6. **核心产品/参数**（产品名+关键指标）
7. **商业化阶段**（是否有收入/客户/订单）
8. **融资情况**（轮次+金额+投资方）
9. **团队背景**（学历/产业经历/专利数量）
10. **军工关联度**（高/中/低 + 证据）
11. **核心优势**（3-5条）
12. **核心风险**（3-5条）

### 3.3 信息填充原则

- 优先用**已有材料**（BP/Teaser）中的数据
- 网页搜索补充融资信息、团队背景
- 有争议或不确认的信息标注"推测""未确认"
- 每个判断尽量注明来源

### 3.4 排序原则

按**商业化成熟度**排序（最成熟的排最前面），让读者先看到最靠谱的公司。

### 3.5 军工关联度的判断标准

| 级别 | 判断依据 |
|------|---------|
| **高** | 有明确军方供货记录、军委项目、涉密机构任务、国防科技进步奖核心成员 |
| **中** | 产品可用于国防、央企军工背景、创始人国防科工背景（但以民用为主） |
| **低** | 消费级/民用为主，军方非核心客户 |
| **未披露** | 公开信息不足以判断 |
